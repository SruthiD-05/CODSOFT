# Calculator Web App

This project is a simple calculator built using **HTML, CSS, and JavaScript** as part of the **CodSoft Web Development Internship**.

## Features
- Addition
- Subtraction
- Multiplication
- Division
- Decimal support
- Backspace function
- Clear button
- Keyboard input support

## Technologies Used
- HTML
- CSS
- JavaScript

## Project Demo
This calculator performs basic arithmetic operations with a clean user interface.
